# Snake-Game

Dependencies:
- ncurses: `system-check` if ncurses comes installed or run `./install.sh` to install dependencies

Clone repo `https://github.com/Puwya/Snake-Game.git`

Change directory `cd Snake-Game`

Make sure you have cmake 3

Run Cmake `cmake ..`

Run EXE `./snake`

# Snake-Game

Dependencies:
- ncurses: going to add script to install if user system does not come with lib

Clone repo `https://github.com/Puwya/Snake-Game.git`

Change directory `cd Snake-Game`

Make sure you have cmake 3

Run Cmake `cmake ..`

Run EXE `./snake`
